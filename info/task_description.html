Enunciado

Lucas, um analista financeiro, está monitorando os preços das ações e quer saber se os preços estão subindo constantemente. Ele precisa de uma função que determine se uma sequência de números é estritamente crescente. Escreva uma função que receba uma lista de inteiros e retorne True se a lista for estritamente crescente, ou seja, cada elemento é maior que o anterior. Para uma lista vazia, retorne True.

Entrada: Uma lista de inteiros.

Saída: Um valor booleano (True ou False).
